Tarjeta Gráfica (GPU)

¿Qué es una tarjeta gráfica?

La tarjeta gráfica o GPU es el componente encargado de procesar y renderizar imágenes, videos y animaciones. Es fundamental para videojuegos, edición de video, diseño 3D e inteligencia artificial.

Componentes clave

  • Núcleos gráficos (CUDA/Stream): procesan operaciones paralelas.
  • VRAM: memoria dedicada para texturas y datos visuales.
  • Frecuencia del núcleo: velocidad de procesamiento.
  • Bus de memoria: ancho de banda para mover datos rápido.
  • Sistema de enfriamiento: aire o líquido.

Tipos de tarjetas gráficas

  • Integradas: vienen dentro del CPU o placa madre.
  • Dedicadas: tarjeta independiente y más potente.

Marcas reconocidas

  • NVIDIA (GeForce / Quadro / RTX)
  • AMD (Radeon / RX series)
  • Intel ARC (nueva competencia)

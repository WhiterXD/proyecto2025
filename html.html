HTML

¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Define la estructura del contenido usando etiquetas, permitiendo agregar texto, imágenes, enlaces, videos y más en un sitio web. No es un lenguaje de programación, sino un lenguaje de marcado que describe elementos de una página.

¿Cómo funciona HTML?

HTML organiza el contenido en elementos y etiquetas. Cada elemento describe una parte de la página, como un párrafo, un encabezado o una imagen. Los navegadores leen este código y lo interpretan para mostrar la página al usuario.

Estructura básica de HTML

<!DOCTYPE html>
<html>
  <head>
    <title>Título de la página</title>
  </head>
  <body>
    <h1>Hola Mundo</h1>
  </body>
</html>

Etiquetas principales en HTML

  • <h1> ... </h1>: Títulos
  • <p> ... </p>: Párrafos
  • <img>: Imágenes
  • <a> ... </a>: Enlaces
  • <ul>, <ol>, <li>: Listas
  • <div>: Organización de bloques

¿Para qué se usa HTML?

  • Crear la estructura de una página web
  • Insertar texto, imágenes y multimedia
  • Crear enlaces entre páginas
  • Definir formularios para usuarios

Consejos al aprender HTML

  • Aprender las etiquetas más comunes
  • Usar indentación correcta para el código
  • Combinarlo con CSS y JavaScript para sitios completos
  • Probar el código en un navegador para ver los resultados

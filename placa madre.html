Placa Madre

¿Qué es la placa madre?

La placa madre es el componente principal de la computadora donde se conectan todos los demás componentes: CPU, RAM, GPU, almacenamiento y periféricos. Funciona como el sistema nervioso del equipo.

Funciones principales

  • Distribuye energía a los componentes.
  • Permite la comunicación entre CPU, RAM y otros dispositivos.
  • Aloja ranuras de expansión como PCIe.

Partes importantes

  • Socket del CPU
  • Slots de RAM
  • Puertos SATA / M.2 para discos
  • PCIe para GPU y tarjetas
  • Chipset que controla funciones del sistema

Factores de forma

  • ATX
  • Micro ATX
  • Mini ITX

CSS

¿Qué es CSS?

CSS (Cascading Style Sheets) es el lenguaje utilizado para dar estilo a las páginas web. Con CSS puedes cambiar colores, tamaños, fuentes, posiciones y el diseño general de un sitio web. Mientras que HTML define la estructura, CSS define la presentación visual.

¿Cómo funciona CSS?

CSS utiliza reglas compuestas por selectores y propiedades. Los selectores indican qué elemento HTML será estilizado, y las propiedades definen cómo se verá ese elemento. El navegador lee estas reglas y aplica los estilos a la página.

Ejemplo básico de CSS

p {
  color: blue;
  font-size: 18px;
}

Formas de aplicar CSS

  • CSS en línea: Dentro de una etiqueta HTML
  • CSS interno: En la sección <style> del HTML
  • CSS externo: En un archivo .css (el más recomendado)

¿Para qué se usa CSS?

  • Dar colores y estilos al texto
  • Definir el diseño de la página
  • Crear animaciones y efectos visuales
  • Hacer que la página sea responsive

Consejos al aprender CSS

  • Entender bien el modelo de caja (box model)
  • Practicar con Flexbox y Grid
  • Aprender sobre responsive design
  • Experimentar con animaciones y transiciones
